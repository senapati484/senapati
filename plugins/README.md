# Plugins

Write custom MCP plugins for Senapati.

## Quick Start

```bash
# Create new plugin
touch ~/.senapati/plugins/my_plugin_mcp.py
```

## Available Plugins

- **telegram_mcp.py** — Send/receive Telegram messages
- **github_mcp.py** — GitHub notifications and PRs
- **spotify_mcp.py** — Spotify playback control

## Contributing

1. Fork the repo
2. Add plugin to `plugins/`
3. Update this README
4. Submit PR
